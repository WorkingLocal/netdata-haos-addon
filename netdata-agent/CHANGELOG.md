# Changelog

## 1.0.0 — Initial release

- Netdata Agent running inside HAOS as an add-on
- Configurable hostname, streaming target, and API key via add-on options
- Supports standalone mode (local dashboard only) and streaming mode (child → parent)
- Monitors host CPU, RAM, disk, network, and system metrics via `host_pid` and `host_network`
- amd64 and aarch64 support
